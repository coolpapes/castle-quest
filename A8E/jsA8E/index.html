Keyboard Controls:
Type normally on the Atari keyboard. Use Ctrl and Shift as modifiers.

Joystick Controls:
• Arrow Keys - Joystick direction (Up/Down/Left/Right)
• Left Alt - Fire button (Trigger 0)
• Right Alt - Trigger 2
• Win/Cmd - Trigger 3

Console Keys:
F2 OPTION • F3 SELECT • F4 START • F5 RESET • F8 BREAK
